Thingを作る

二つのThing - s1とs2をつくる

% aws iot create-thing --thing-name s1
{
    "thingName": "s1",
    "thingArn": "arn:aws:iot:ap-northeast-1:533718670665:thing/s1",
    "thingId": "f244d339-0a9a-4407-be76-757e40da0acd"
}
% aws iot create-thing --thing-name s2
{
    "thingName": "s2",
    "thingArn": "arn:aws:iot:ap-northeast-1:533718670665:thing/s2",
    "thingId": "a48971d4-423f-4b66-aee1-ac63daae3892"
}

確認する

aws iot list-things

キー関係を取得してそれぞれのディレクトリへ置く

% mkdir s1    
% mkdir s2
