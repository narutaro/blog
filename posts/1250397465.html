TOOL BOX

Jul 9th, 2022

apt command

apt list --installed
apt-cache search <keyword>
apt-get install

Check global IP address

curl inet-ip.info

Named profiles for the AWS CLI

Setting a profile

aws configure --profile <profile_name>

The setting will be stored in ~/.aws/credentials

Use a profile

export AWS_PROFILE=<profile_name>

Local web server

python3 -m http.server --bind 192.168.68.104

Create a large file

base64 /dev/urandom | head -c 128000

Cloudwatch logs

You can tail -f for the Amazon CloudWatch logs.

Lambda logs

aws logs tail --follow /aws/lambda/<function_name>

You don't have to put full log group name. Just hit tab after /aws to complement the rest.

IoT Core logs

aws logs tail --follow AWSIotLogsV2

SCP

Copy a remote folder to local

scp -r ubuntu@host_name:/path/to/the/dir .
